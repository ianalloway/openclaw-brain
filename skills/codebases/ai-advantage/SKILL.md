# ai-advantage

## Overview

AI Advantage Edge sports betting site and tech. Sports betting platform with ML predictions, Kelly Criterion bet sizing, and value bet detection.

**Location:** `https://github.com/ianalloway/ai-advantage`
**Status:** Active

## Tech Stack

- **Framework:** React (Vite)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Package Manager:** Bun

## Dependencies

See `package.json`:
- react, react-dom
- tailwindcss
- Sports betting ML integration
